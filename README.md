# impact_social

Concevoir, construire et maintenir des solutions digitales qui favorisent l'économie sociale.

## Projet en cours : Radar Citoyen

Protocole B2G d'ownership citoyen des murs commerciaux : fractionnement foncier, crowdfunding de travaux, financement par royalties indexées sur le chiffre d'affaires des commerçants, sortie automatique à un multiple contractuel.
